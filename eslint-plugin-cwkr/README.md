# ESLint Plugin

## Installation

```bash
$ yarn add --dev eslint-plugin-cwkr
$ npm install --save-dev eslint-plugin-cwkr
```

## Usage

```json
{
    "extends": [
        "eslint:recommended",
        "plugin:@typescript-eslint/recommended",
        "plugin:cwkr/recommended"
    ]
}
```
